# StudyFlow

StudyFlow adalah aplikasi mini fullstack untuk membantu mahasiswa mengatur jadwal dan task belajar dengan UI yang modern dan responsif.

## Tech Stack
* **Frontend:** Next.js 15 (App Router), Tailwind CSS, Lucide React
* **Backend:** Express.js, Node.js, CORS
* **Database:** In-memory Array (sementara)

## Cara Menjalankan Backend Secara Lokal
1. Buka terminal dan masuk ke folder backend: `cd backend`
2. Install dependencies: `npm install`
3. Jalankan server: `npm run dev`
4. Backend akan berjalan di `http://localhost:5001`

## Cara Menjalankan Frontend Secara Lokal
1. Buka terminal baru dan masuk ke folder frontend: `cd frontend`
2. Install dependencies: `npm install`
3. Jalankan aplikasi: `npm run dev`
4. Frontend akan berjalan di `http://localhost:3000`
